# Projets du portfolio

Cette section regroupe les projets data sélectionnés pour démontrer mes compétences techniques, métier et comportementales dans le cadre du portfolio Aéroworld.

Chaque projet est présenté comme une preuve concrète de compétences acquises : analyse de données, data quality, modélisation, dashboarding, machine learning, aide à la décision, communication et posture de conseil.

Les projets ont été choisis pour couvrir des contextes variés — santé publique, immobilier, e-commerce, gestion de projet, analyse client, export international et fraude financière — afin de montrer la polyvalence de mon profil Data Analyst / BI & Analytics Engineer.

## Logique de présentation

Chaque fiche projet suit une structure homogène :

- contexte métier ;
- objectifs du projet ;
- données utilisées ;
- méthodologie ;
- outils et technologies ;
- résultats obtenus ;
- compétences démontrées ;
- limites et pistes d’amélioration.

Cette organisation permet de relier chaque projet aux attentes du client Aéroworld et de transformer le portfolio en support de démonstration par preuves.
